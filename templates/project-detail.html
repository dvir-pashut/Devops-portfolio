the main tools for this project

this project present a rest api that is served in the cloud and include ci-cd testing and more

  • Ci - github action
  • notifications - slack
  • cloud - gcp
  • deployment - k8s
  • database - mongodb
  • rest api - Flask(python)
  • logging - EFK(elastic, fluentd, kibana)
  • monitoring - prometheus and grafana
  • certification - let's encrypt
  • IAC - terraform
  • secrete managment - bitnami sealed secretes
  • gitops - argocd
  • static files - nginx
